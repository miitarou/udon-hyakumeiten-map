## Review Notes

This app is an unofficial personal reference map built from publicly available "うどん・そば百名店" related information.
It is not affiliated with, sponsored by, endorsed by, or approved by Tabelog, Kakaku.com, or 食べログ百名店.

The app does not collect accounts, analytics, review text, ratings, comments, or photos.
Location permission is used only when the user explicitly searches for restaurants around the current location.
Location data is used locally for map display and distance calculation, and is not sent to or stored on any server.

Visit status such as "行きたい" and "訪問済み" is stored locally on the device.
Export/import uses the user's file/share action and is not uploaded by this app.

Restaurant data is bundled with the app for offline startup.
When online, the app may fetch the latest public JSON data from the GitHub Pages site for data freshness.
HTML, CSS, and JavaScript are not remotely replaced; feature changes are shipped through App Store updates.

External restaurant links open in Safari or the platform browser.
