# App Store Assets

App Store申請用の素材置き場です。

## Screenshots

`screenshots/` には、iPhone Pro Max系サイズ想定の 1290 x 2796 PNG を置いています。

- `01-map-overview.png`: 地図全体とクラスタ表示
- `02-search-sheet.png`: 検索・リストのボトムシート
- `03-filters-expanded.png`: 絞り込みフィルタ
- `04-search-results.png`: 店舗検索
- `05-store-popup.png`: 店舗ポップアップ
- `06-recommendations.png`: 「この店が好きなら」推薦
- `contact-sheet.png`: 確認用一覧。App Store提出用ではない

現在のスクリーンショットはWeb版をiPhoneサイズでレンダリングした申請候補です。
Xcode / Simulator環境が整った後、実アプリ画面で最終確認し、必要に応じて差し替えます。
